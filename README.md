# Template pour dev de css

Préalables:
- Nodejs
- npm

On assume que l'utilisateur a déjà sa clé ssh chez bitbucket.

## Pour commencer
Le setup:
``` 
git clone git@bitbucket.org:uqam/esg_gabarit.git
cd esg_gabarit
npm install
```

## Créer le fichier css

```
gulp sass2css
```
